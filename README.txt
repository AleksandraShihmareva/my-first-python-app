Приложение с функциональностью записной книжки.
Использован python 2.7 + django + небольшой скрипт на JQuery для реализации всплывающего окна. База данных - sqllite.

Во всплывающем окне отображаются пользователи, у которых день рождения в текущий день. Окно появляется только один раз при входе в приложение (так сделано специально, чтобы окно не всплывало каждый раз при редактировании данных таблицы). Для повторного отображения окна нужно выйти и снова залогиниться.
